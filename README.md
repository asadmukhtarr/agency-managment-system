# 📌 Overview

Agency ERP & CRM Management System is a modern, enterprise-level business management platform developed for Digital Agencies, Software Houses, IT Companies and Freelancers.

The application centralizes CRM, Project Management, Client Management, Staff Management, Quotations, Estimates, Payments, Portfolio, Products, Services, Blog CMS and Company Settings into one powerful dashboard.

The system helps agencies streamline business operations, improve productivity, manage projects efficiently and track client activities from a single admin panel.

# 🚀 Installation

```bash
git clone https://github.com/yourusername/agency-erp-crm.git

cd agency-erp-crm

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan storage:link

npm run build

php artisan serve
```
